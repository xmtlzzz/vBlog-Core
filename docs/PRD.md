# vBlog Core

## 项目背景

目前开源的博客系统很多，但是我想要实现一个可以让用户自定义博客组件的开源博客项目，实现轻量部署、方便使用的特点。

同时预留gRPC接口去和client软件工具对接，实现博客系统的数据采集以及基本信息展示例如打开客户端exe可以显示博客的信息、新增关注等内容。

## 技术选型

1. 前端：React
2. 后端：Go Restful框架、gRPC接口支持
3. UI需求：简洁明了、浅色调，可以支持手动/根据系统切换明暗模式
   1. 采用Element UI进行样式布局
4. UX需求：动画切换丝滑、加载动画支持
   1. 报错状态显示：失败、成功分别展示不同的颜色，并且效果生动
5. 数据存储：Postgresql
   1. 每个模块都单独一张表进行数据记录
   2. 按需构建索引
6. Blog Server和客户端之间通过预留的gRPC接口通信，主要实现：
   1. 客户端感知博客列表的变化
   2. 文章关注信息可以通知到客户端
   3. 采集Go wails框架构建client，client的前端ui按需选择，同样遵循简洁轻量化需求
7. Blog支持单独的后台管理页面，博客内容主要是markdown格式的渲染器

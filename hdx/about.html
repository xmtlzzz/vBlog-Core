vBlog Core

一个极客的博客实验 — 用代码写作，用文字思考

这是什么

vBlog Core 是一个为 vibe coder 设计的可自定义博客系统。不是又一个静态博客生成器，而是一个真正轻量、前后端分离、支持组件自定义的动态博客。

你可以用 Markdown 写作，用 React 自定义 UI 组件，用 gRPC 实时同步内容，用 Wails 把它打包成桌面客户端。

设计理念

每个模块单独一张表，索引按需构建。组件可自定义，UI 按需选择。不追求大而全，只追求恰到好处。

vBlog 的核心原则是克制：不引入不需要的依赖，不过度设计数据结构，不替用户决定博客应该长什么样。你想要什么样的博客，就用 React 组件去实现它。

这是一个持续演进的项目。代码开源，欢迎 PR。
